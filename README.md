## Про проект

Дипломний проект. Система управління технічним обслуговуванням 
автопарку підприємства харчової промисловості.

**Автор:** Сінько Севастьян  
**Рік:** 2026  
**Платформа:** WordPress + WP Data Access + MySQL 8.4

## Функціонал

- Автоматичне отримання пробігу від GPS-трекера через REST API
- Система нагадувань «Світлофор» з 4 рівнями терміновості
- Наряди на ТО з Dependent Lookup для робіт і запчастин
- Складський облік з автоматичним списанням через MySQL тригери
- 4 типи звітів на основі MySQL VIEW
- Розмежування доступу для 4 ролей (RBAC)

## Структура бази даних

22 таблиці в 3 рівнях:
- **Рівень 1:** 12 довідникових таблиць
- **Рівень 2:** 5 основних таблиць
- **Рівень 3:** 5 операційних таблиць

## Встановлення

1. Встанови WordPress 6.x
2. Встанови плагіни: WP Data Access, Code Snippets, User Role Editor
3. Імпортуй `database/automaint_db.sql`
4. Виконай `database/triggers.sql`
5. Виконай `database/views.sql`
6. Активуй плагін `plugin/fleet-manager.php`
7. Додай сніпети з папки `snippets/` через Code Snippets
